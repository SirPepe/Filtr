Filtr
=====

Simplistic but performant filters for canvas. MIT license.

How to use
----------

    filtr(canvasElement).sepia(50).saturation(80).brightness(20).render();

API reference
-------------

* `filtr(canvasElement)`
* `filtr.brightness(amount)`
* `filtr.contrast(amount)`
* `filtr.saturation(amount)`
* `filtr.sepia(amount)`
* `filtr.updateSource()`
* `filtr.render()`
